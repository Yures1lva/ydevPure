Desenvolvimento do site MKT Criativo.

Contextualização

O site Marketing Criativo é um projeto de uma agencia de Marketing fictícia criada apenas para portfólio, todo o design contido neste projeto foi criado do zero (com exceção das ilustrações) com base em conceitos de Interface de Usuários e Experiência de usuários, podendo se utilizar do site em qualquer dispositivo. Aqui será destacado as principais informações sobre como se deu o desenvolvimento do site.

Ferramentas

Para o desenvolvimento do site foi utilizado as três bases do desenvolvimento web, html, css e Js não foi utilizado nenhuma ferramenta externas, tentando implementar os principais conceitos dessas tecnologias aplicando responsividade, semântica e controle do DOM.

Html

Abordando as principais técnicas de acessibilidade, o html do site foi utilizado todos os conhecimento de semântica do desenvolvedor, sem a utilização de “gambiarras” no decorrer do desenvolvimento, seguindo uma hierarquia bem aplicada com nomenclaturas de classes bem definidas e fáceis de identificar.

CSS

Padronizando cores e fontes em variáveis para possíveis futuras mudanças, o layout do site foi bem estruturado organizando os styles em pastas separadas onde componentes reutilizáveis ficaram de fácil acesso bi arquivo principal de css, foi implementado dois breakpoints para responsividade ultrawide e telas pequenas.

JavaScript

Foi utilizada uma paleta com variações de roxo vibrante que lembra a modernidade na arte e o conceito de maximalismo na escolha da paleta, junto a cor primária foram desenvolvidas cores secundárias apenas mudando elementos como brilho, intensidade ou sombreamento da primária.

Resultado Final

O protótipo conseguiu chegar a um resultado satisfatório com um design elegante seguindo um padrão bem definido com padrões de Design de Interface e User Experience aplicados ao contexto do site, foi realizado a prototipação também para a versão mobile dando mais escalabilidade par ao projeto.



Conclusão

O projeto conseguiu entregar sua premissa, um site de apresentação de uma empresa de Marketing de modo que apresentando os detalhes da empresa com um visual agradável e dinâmico.

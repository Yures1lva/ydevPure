Desenvolvimento da MF produtora.

Contextualização

Com o design system criado e acordado com o cliente, foi possível partir para o desenvolvimento do site, neste caso foram escolhida novas ferramentas de desenvolvimento, alguns detalhes foram acertados com o cliente, como a cor principal e design de algumas imagens, mas fora isso o desenvolvimento seguiu a risca tudo que foi implementado no design da interface do site.

Ferramentas

Nesta aplicação foi utilizado tecnologias mais atuais, pensando em como o site pode evoluir pra mais páginas futuramente foi utilizado como ferramenta principal o ReactJs um framework/biblioteca de JavaScript de desenvolvimento web, uma ótima aplicação para controle inteligente do frontend por meio de components, junto a essa tecnologia foram utilizadas alguns pacotes conhecidos por agilizar e aumentar a performance da aplicação em desenvolvimento, o typeScript e o vite.

Components

Das inúmeras vantagens de se usar o react uma de bastante destaque para o frontend é a utilização do conceito de components, se dá basicamente pelo fato do reaproveitamento de código, diminuindo drasticamente o tamanho de sua aplicação, onde elementos que se repetem na tela podem ser facilmente reutilizados mudando apenas seus atributos se necessário. Com o uso dessa ferramenta o desenvolvimento do se deu de forma mais organizada e eficiente.

TypeScript

O uso dessa ferramenta foi necessário para melhorar a tipagem do projeto, com o TS é possível adicionar recursos que não estão presentes de maneira nativa em JavaScript, essa ferramenta fará bem mais diferença com a possível expansão do site futuramente.

Vite

A utilização do vite se deu para diminuir a sobrecarga de configurações no início do projeto, o vite consegue configura toda a comunicação entre as partes do projeto, as partir de sua utilização o desenvolvedor passa a se preocupar apenas em criar o css e o html da aplicação já que o JavaScript pode ser executado em um mesmo documento graças as configurações de projeto realizadas pelo vite+React.

Resultado Final

Após ser desenvolvido e apresentado para o cliente. foram feitas algumas alterações básicas de design, contudo o cliente se sentiu satisfeito e o site pode ser entregue, o site está no ar apenas com uma ferramenta de deploy, assim que o cliente resolver algumas questões, ele poderá ser hosteado em seu domínio..



Conclusão

O desenvolvimento com ferramentas mais atuais serviu para se conhecer mais sobre eficácia no desenvolvimento, o próximo passo é criar uma página de adm que fique responsável por gerenciar os dados do site, como clientes, resultados ou feedback, mas para o que foi solicitado pelo cliente, apenas um site de apresentação este projeto conseguiu chegar em seu resultado esperado.
